# Scripts

Deployment and management scripts for Percolate.

## Solana

```bash
./scripts/deploy-solana.sh devnet
./scripts/deploy-solana.sh mainnet
```

## EVM

```bash
./scripts/deploy-evm.sh ethereum
./scripts/deploy-evm.sh base
./scripts/deploy-evm.sh arbitrum
```

## Initialize

```bash
./scripts/init-protocol.sh
./scripts/register-collateral.sh USDC
./scripts/register-collateral.sh SOL
```
